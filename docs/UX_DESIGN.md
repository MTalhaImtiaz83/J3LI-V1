# J3LI Discovery Interface: The Constellation Model

## Core Philosophy: "Architected for the Hungry"
This interface moves away from broken flat text and into a "Living Network" of knowledge.

## Visual Design System
- **Primary Canvas**: Void Navy (Deep depth).
- **Divine Thread (Quran)**: Thin Gold energy lines connecting verses.
- **Scholarly Context (Hadith/Seerah)**: Faint Emerald branches.
- **Galactic Backdrop**: 6,236 verses as distant stars to show scale.

## Navigation & Controls
- **Whoosh Navigation**: Camera centers on clicked nodes with smooth animation.
- **Luminescence Hover**: Nodes glow when the camera is near, solving contrast issues.
- **Tethering**: Visual breadcrumb lines showing the search path.
- **Esc Key**: Reset to full constellation view.
- **Right-Click**: Step back one level.
- **Space Bar**: Return to main home/search screen.

## Story Mode
Guided journeys through the Quranic landscape, automatically flying the user through chronological sequences of revelation.
